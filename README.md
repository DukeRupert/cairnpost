# CairnPost

A lean, handcrafted CRM for small service businesses. Part of the Firefly Software product suite.

CairnPost helps small service businesses manage their lead-to-close pipeline — contractors, consultants, tradespeople, and similar relationship-driven businesses. It serves as the contact and deal layer shared across the Firefly product suite.

Built by [Firefly Software](https://fireflysoft.dev) in Helena, Montana.

## Stack

- **Go** — server
- **htmx + Alpine.js** — interactivity
- **Svelte** — high-interactivity islands (pipeline kanban)
- **Tailwind CSS** — styling
- **templ** — HTML templating
- **PostgreSQL** — database
- **Postmark** — transactional email
- **Docker Compose** — infrastructure
- **Caddy** — reverse proxy

## Status

In development. Not yet accepting users.
