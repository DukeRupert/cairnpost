# CairnPost

Forms → Contacts → CRM. Sort the gold from the gravel.

CairnPost helps small businesses collect customer inquiries through simple forms and turn them into a clean, validated contact list. Every form submission is a potential contact — the business owner triages what comes in, accepting real inquiries and discarding spam, building their customer list over time.

Built by [Firefly Software](https://fireflysoft.dev) in Helena, Montana.

## Stack

- **Go** — server
- **htmx + Alpine.js** — interactivity
- **Tailwind CSS** — styling
- **templ** — HTML templating
- **PostgreSQL** — database
- **Postmark** — email notifications
